  • es6中新增的几个数组方法

    2017-12-01 |
    • { es6 }
    /
    • { 前端 }

    forEach()

    forEach遍历数组，无返回值，不改变原数组，仅仅只是遍历、常用于注册组件、指令等等。

    map()

    map用于遍历数组，返回一个新的数组，不改变原数组的值

    filter()

    过滤某个条件的数组，不改变原来的数组的值

    reduce()

    让数组的前后两项进行某种计算。然后返回其值，并继续计算。不改变原数组，返回计算的最终结果

    some()

    //遍历数组每一项，有一项返回true,则停止遍历，结果返回true。不改变原数组

    every()

    遍历每一项，如果每一项都返回true时，最终返回true

    总结： 现在基本项目都是在webpack下开发的，用es6的新语法只需要配置相关的解析loader就是可以了，还算是比较方便的

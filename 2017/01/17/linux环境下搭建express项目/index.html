linux环境下搭建express项目

发表于 2017-01-17 |

导语

在node项目中，我们大多数情况下是在linux下运行我们的node项目，这里记录一下如何在本地的虚拟机中安装linux并且搭建一个居于express的node项目

工具

  • 虚拟机使用oracle的virtualBox
  • centos 7官网下载地址
  • xshell 5官网下载地址
  • ps:可以把xftp也顺便安装一下 xftp官网下载地址
  • sublime text3的sftp插件安装一下，直接用package包管理工具安装即可

安装虚拟机和linux

创建一个虚拟机

选择一个虚拟机镜像文件

设置网络类型，如果是路由器上网，直接设置成桥接网卡

选择安装centos 7

期间选择默认的英文，创建用户名密码，是否安装GUI还是basic模式，扩展包可以选择开发工具进行安装，选择安装的硬盘等信息。

在xshell中建立对linux的连接

先来启动网卡

vi /etc/systemfig/network-scripts/ifcfg-enp0s3

将ONBOOT设置成yes
ps：操作，按住i进去编辑模式，修改之后按：切换成命令模式，输入wq，保存退出即可

修改之后重启网络

systemctl restart network

查看IP地址

ifconfig

可以看到

在xshell中建立连接


这样我们可以直接在xshell中操作我们的linux系统

在linux中安装node

想安装epel和nodejs，通过yum来安装 

yum install epel-release 
yum install nodejs

测试node是否成功安装 node -v

创建express项目

这里适用express的cli来快速创建项目，当然也可以不使用

首先安装express-generator ，并且用起来执行express安装命令

npm install -g express-generator
express test02
npm install    //切换到test02目录

创建完成

运行我们的项目 

npm start   也可以执行    node bin/www

新建窗口 执行 curl http://localhost:8080测试是否输出一段html标记

在物理机上访问项目

先关闭一下linux的防火墙

systemctl stop firewalld

访问

在sublime中通过之前下载好的sftp插件下来到物理机上进行编辑，并设置配置参数，实时有变动时提交到服务器中

# node # express

搭建公司内网npm

由于公司的很多前端组件不方便公开到npm中，也为了给各位前端员工互相调用，分享，搭建公司内网的npm成为了必要
截图如下：

使用工具

  • sinopia

遇到问题

  • 网络问题
  • 设置代理没有用，清空代理，清空缓
  • npm cache clean

配置文件

  • vi /root/.config/sinopia/config.yaml

储存文件

  • /root/.local/share/sinopia/storage

公司内网访问

  • http://192.168.1.3:4873/

服务端安装pm2守候进程

以下为客户端操作

  • 安装内网npm
  • 查看在使用的npm
  • 使用某一个npm

在linux中添加用户(用户名，密码，邮箱)

  • npm adduser

客户端登陆

  • npm login

客户端发布

  • 发布之前注意每一次发布的版本号不一样，在package.json中version参数
  • npm publish

客户端退出

  • npm logout

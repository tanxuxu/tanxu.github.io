css高级技巧

发表于 2016-09-03 | 分类于 html/css/js |

导语

列举一下常见的css技巧，在项目中比较常用，有些css技巧你不一定要能完整的写出来，用的时候知道有这个东西，写不出来可以马上能花几秒时间谷歌出来的也是可以的。比较小三角我经常忘了怎么写

排除最后一个元素做某个样式

我们经常为li,dt,dd,a等等加右边框以达到一种相隔的样式效果，但是往往最后一个是不需要的，下面的图你是不是经常做

上面的图每一个dl有个右边框，最后一个没有
按照正常思维你可能会这样

li{
     border-right:1px solid #ddd;
}
li:last-child{
    border-right:none;
}

或许你有更好的方式去做这件事

li:not(:last-child){
    border-right:1px soild #ddd;
}

不截取字符串的情况下显示几行，剩下的显示省略号

如图：

像这样的手机页面，要适应的分辨率不同的手机，怎么做到在小屏幕上显示一行文字，怎么在中等屏幕上显示2行，大屏幕显示三行，如果你做个filter去做也没问题，但是能不用js的我们尽量不去用。那么css上怎么去做这件事呢

h2{
    line-height: 18px;
   word-break: break-all;
   overflow: hidden;   /*必须*/
   text-overflow: ellipsis;  /*后面的三个点*/
   display: -webkit-box;   /*必须*/
   -webkit-line-clamp: 2;   /*保留多少行*/
   -webkit-box-orient: vertical;  /*必须*/
   font-size: 14px;
   font-weight: 700;
}

这样的话在小屏幕的话你要显示一行可以这样

@media screen and (max-width: 359px)
.h2 {
    font-size: 12px;
    -webkit-line-clamp: 1;
}

css去做一个小三角

这个相信很多人都知道，就不多说了，直接贴代码，方便自己不记得时直接过来拷贝

向上的

.mainNav .sideNav>i {
    position: absolute;
    top: 15px;
    right: 43px;
    border-left: 4px solid transparent;
    border-right: 4px solid transparent;
    border-bottom: 4px solid #fff;
    display: inline-block;
    width: 0;
    height: 0;
    overflow: hidden;
    font-size: 0;
    line-height: 0;
}

向下的

.mainNav .sideNav>i {
    position: absolute;
    top: 15px;
    right: 43px;
    border-left: 4px solid transparent;
    border-right: 4px solid transparent;
    border-top: 4px solid #fff;
    display: inline-block;
    width: 0;
    height: 0;
    overflow: hidden;
    font-size: 0;
    line-height: 0;
}

琢磨两遍你就知道规律了，向左向右的可以推出来怎么写

垂直居中的方式方案

单行文本垂直居中

这个不用多说，都会的

h2{
    height:200px;
    line-height:200px;
}

通过绝对定位是居中

这个也不过多少，都会的

#parent {
    position:relative;
}
#child{
    width:100px;
    height:100px;
    position:absolute;
    left:50%;right:50%;
    margin-left:-50px;
    margin-top:-50px;
}

图片在某个盒子中垂直居中

<div id="parent">
    <img></img>
</div>

#parent img{
  vertical-align:middle;
}

某个盒子在另一个盒子里面垂直居中

<div id="parent">
    <div id='child'></div>
</div>

#parent{
    display: table
}
#child{
    display: table-cell;
    vertical-align: middle;
}

table等宽设定

table{
 table-layout: fixed;
}

如何让整个网站的图片会灰色

有的时候你不想要彩色的网站了，试图让图片都变成灰色

img{
    filter: grayscale(100%);
    -webkit-filter: grayscale(100%);
}

巧用css3的calc计算属性

注意运算符两边留一个空格，否则可能识别不了
像素单位记得带上

div{
 width: calc(100% - 100px);
}
